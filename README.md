# Dharma Delight School Website

Frontend for a modern Indian school website with Hindu cultural design language, bilingual support, and responsive pages.

## ⚠️ Important Contribution Rules

- Direct pushes to the `main` branch are not allowed.
- Always create a separate feature branch before making changes.
- Submit changes through a Pull Request (PR) only.

### Recommended Workflow

```bash
git checkout -b feature/your-feature-name
git push origin feature/your-feature-name
